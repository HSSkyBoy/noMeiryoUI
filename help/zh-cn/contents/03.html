3.更新记录

2024/03/05 版本 3.3.1

  • 更新了簡体中文（中国）的翻译。(zh-CN.lng,zh-CN.chm)
    感谢 kingofotaku。
  • 更新了繁体中文（台湾）的翻译。(zh-TW.lng)
    感谢 abc0922001。

2024/02/25 版本 3.3.0

  • 现在您可以修改本软件的字体了。
    此外，还修改了显示语言为日语时的默认字体。
  • 为防止意外错误，现在您可以选择是否允许多重启动本软件。
  • 更新了繁体中文（台湾）的翻译。(zh-TW.lng)
    感谢 jay900604。

2023/09/03 版本 3.2.1

  • 更新了繁体中文（台湾）的翻译。(zh-TW.lng)
    感谢 abc0922001。

2023/08/27 版本 3.2.0

  • 使软件语言跟随Windows显示语言
  • 修正了在诸如Windows 11 记事本等程序中出现框架线条等的显示错误
  • 由于以上修正而将最低支持版本提升至 Windows Vista。
  • 现在，在导入字体配置时将会使各字体的大小设置为系统的初始值，以适应当前界面的DPI。
  • 修正了当更改标题栏的字体大小时导致标题栏可能因适应字体而变得过高的问题。

2022/09/26 版本 3.1.0

  • 采用 Visual C++ 2008 进行构建以避免杀毒软件误报。

2022/09/21 版本 3.1.0

  • 在 Windows 11 2022 Update(22H2) 中，出于避免在将标题栏设置为透明时产生问题，故强制将标题栏的字体固定为系统默认字体。
  • 出于上述原因，如今在 Windows 11 2022 Update(22H2) 中已无法选择并更改标题栏字体。。
  • 在 Windows 11 2022 Update 中使系统版本得以正确显示。

2022/06/26 版本 3.0.0.1

  • 加入了由 Luke Luo 提供的简体中文翻译及新的字体显示设定

2022/02/13 版本 3.0.0

  • 现在，配置文件中的字体大小以 pt 为单位。
    因此，第二版和第三版程序的字体配置文件不可互相替代。
  • 现在，语言资源文件用 UTF-8 编码。
  • 现在，设置按钮的文字更易于理解。

2021/11/28 版本 2.42(目前只在 GitHub 上发布)

  • 添加了 Windows 11 的系统默认字体。
  • 现在，Windows 11 的版本号会正常显示了。

2021/01/10 版本 2.41

  • 现在能够读取各字体下所有的字形了。
  • 支持显示Windows 10 Version 20H2的版本号。
