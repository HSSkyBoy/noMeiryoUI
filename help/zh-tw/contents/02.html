按下「設定」按鈕，你就能設定使用者介面字型。

Options menu

  • Choice No!! Meiryo UI application font
    Users can choice application font by font dialog.
  • Set system fonts in separated thread
    Some programs cause system freeze when system fonts changed.
    To avoid system freeze check this item.
    Initially, this option is checked.
    If you don't select this item, you can reduce time when you changed system fonts.
  • Calculate font size same as Windows 7
    In Windows 8, font size caliculate algorithm is different from other version of Windows.
    When you check this option, font size caliculate algorithm in Windows 8 is same as other version of Windows.
  • Don't run multiply
    When you check this option, No!! Meiryo UI don't run multiply.
    This option is useful to avoid contamination.
  • Force title bar font change enabled (no warranty)
    On Windows 11 22H2 and later, enables title bar font change button when you check this option.
    On Windows 11 22H2 and later, when you try to change title bar font, unknown troubles may occurred.
    This is why, this option is no warranty.

命令列選項。

命令列格式如下：
noMeiryoUI [font_setting_file] [options...]
If you use multiple options you must separate option name by space.

如果你指定了「font setting file」，字型設定檔將會在程式啟動時載入。

Options

In [options...], you can use these options.
If you use multiple options you must separate option name by space.

  • -set
    　-set option can set systems fonts on start up and exit program.
    This option is useful for startup shortcut, launcher setting, batch file.
  • -delay
    　-delay option delays system fonts change by seconds.
    You must specify delay time be seconds. For example, -delay 60.
    You can use this option to prevent system changes system fonts in some reasons
  • -force (no warranty)
    　-force option enables to change title bar font on Windows 11 22H2 and later.
    On Windows 11 22H2 and later, when you try to change title bar font, unknown troubles may occurred.
    This is why, this option is no warranty.
